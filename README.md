## Getting Started

- You need MongoDB and NodeJS for this project
- Config enviroment variable with .env.local

#### 1. Install

```bash
# Dependencies Install
npm install
# or
yarn install
```

#### 2. Run

- development

```
yarn dev
```

- production

```
yarn start
```
